Creación de usuario

Universidad de La Laguna

Creación de usuario:

Para poder Identificarnos en Github deberemos previamente haber creado un usuario desde la web principal. Deberemos seguir las instrucciones indicadas para finalmente definir nuestro usuario en Github. Github permite crear dos tipos de plan para nuestras cuentas de usuario:

  • Cuenta pública
  • Cuenta de pago
    • Personal
    • Organización
    • Empresa

La principal diferencia entre una cuenta pública y otra de pago radica en que esta última permite establecer la visualización de nuestros repositorios en privado, lo que da una mayor privacidad al desarrollo de nuestros proyectos. Dentro de las cuentas de pago, tenemos la posibilidad de escoger entre distintos planes, dependiendo de las características requeridad o el entorno de trabajo.
